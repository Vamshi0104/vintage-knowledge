Introduction to Datatypes

Definition

≫A Data Type is a classification that specifies which type of value a variable has and what type of mathematical, relational or logical operations can be applied to it without causing an error.For example, string is a data type that is used to classify text..
≫Each datatype has different storage capacities.
 ⇒ So,C language supports 2 different type of data types:

I ) Primitive Data_types ----≫ char , int , float , double , void

II ) Derived Data_types ----≫ array , stucture, union and pointer

# Additional three more data types have been added in C99:
  • _Bool
  • _Complex
  • _Imaginary

# Memory storage differs from 32 bit Architecture to 64 bit Architecture.
 ⇒For example,the size of "int" keyword is 2 bytes in 32 bit whereas 4 bytes in 64 bit ...

char

Definition

≫ char is keyword in C used for declaring character type varaibles.

Syntax

char  ;

Some Points to be known about char

≫ The size of the character variable is 1 byte.

≫ For example,Initialization of character type varaiable char p = 'y' ;

≫ We can store only one character using character data type.

≫ The Range of character type varies from -128 to 127 ( signed char ) and 0 to 255 (unsigned char).






int

Definition

≫ int is keyword in C used for declaring integer type varaibles.

Syntax

int <variable_name> ; // '< >' indicates variable name is mandatory.

Some Points to be known about int


≫ Integers are whole numbers that can have both zero, positive and negative values but no decimal values are included.

≫ For example, int a = 5; is initialization of integer variable.The storage size of int data type is 2 or 4 or 8 byte which depends on the Architecture used (16 bit or 32 bit or 64 bit).

≫In C programming we can declare multiple variables at once. For example , "int a,b;"

≫ If we want to use the integer value that crosses the above limit(range), we can prefer for "long int" and "long long int" by using long keyword for which the limits are very high.


float and double

Definition

≫float and double are keywords in C that are used to hold decimal values of integer that is hold real number.

Syntax

float <variable_name>;

Some Points to be known about float and double


≫ The size of float(4 bytes), double(8 bytes), long double(16 bytes) ,short(2 bytes) ,long long(8 bytes) is fixed and idependent of Architecture used(32 bit or 64 bit).Floating-point numbers are also represented in exponential format.
For example, float t = 3.4e3

≫Difference between float and double ??

 → The size of float (Single Precision float data type) is 4 bytes. And the size of double (Double Precision float data type) is 8 bytes.

void

Definition

≫ void datatype actually refers to an object that does not have a value of any type

Some Points to be known about void


≫ void is an incomplete type.

≫ Generally void is an empty data type and it used as a return type of functions that return no value in c.
For Example :
→void func(int n) //It indicates that the function returns no result.
→int func(void) //It indicates that the function has no parameters.

≫ One thing that you must remember is if a function is not returning anything, it means that return type is void

≫ Therefore it is not possible to create variable of void type
